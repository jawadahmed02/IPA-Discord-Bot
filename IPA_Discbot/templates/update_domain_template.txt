### New Predicates
```text
- (at ?r - robot ?l - location): robot r is at location l
- (carrying ?r - robot ?p - package): robot r is carrying package p
- (delivered ?p - package ?l - location): package p has been delivered to location l
- (connected ?l1 - location ?l2 - location): locations l1 and l2 are connected
```

### FUNCTIONS
```text
- (battery-level ?r - robot) - number: current battery level of robot r
- (distance ?l1 - location ?l2 - location) - number: distance between locations
```

### Action Parameters
```text
- ?r - robot: the robot performing the action
- ?from - location: starting location
- ?to - location: destination location
```

### Action Preconditions
```lisp
(and
  (at ?r ?from) ; robot must be at starting location
  (connected ?from ?to) ; locations must be connected
)
```

### New Predicates
```text
- (visited ?r - robot ?l - location): robot r has visited location l
```

### Action Effects
```lisp
(and
  (not (at ?r ?from)) ; robot is no longer at start
  (at ?r ?to) ; robot is now at destination
  (visited ?r ?to) ; mark location as visited
)
```
